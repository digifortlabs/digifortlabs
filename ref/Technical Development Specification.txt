Digifort Labs: Technical Development Specification

Version: 2.2 (Database & API Implementation Edition)
Stack: React, Node.js, PostgreSQL, AWS (Mumbai)
Tenant Logic: Logical Isolation (Multi-tenant)

1. System Architecture Overview

The system is designed as a cloud-native, multi-tenant application hosted in the AWS ap-south-1 (Mumbai) region to ensure data sovereignty.

1.1 High-Level Stack

Frontend: React 18+ (Tailwind CSS, Lucide Icons).

Backend: Node.js (Express).

Database: Amazon RDS (PostgreSQL).

Storage: Amazon S3 with OAC (Origin Access Control).

CDN: Amazon CloudFront (Restricted to India IP ranges).

2. Multi-Tenancy & Data Isolation

2.1 Database Level

Every table (Patients, Records, Users) contains a hospital_id UUID column.

Row-Level Security (RLS) or mandatory WHERE clauses must be enforced in the DAO layer.

2.2 Storage Level (S3)

Path Structure: s3://digifort-labs-mrd-india/hospitals/{hospital_id}/{patient_id}/{file_id}.pdf.gz

Access: Direct S3 access is blocked. Files are served via CloudFront using Signed URLs with a 15-minute expiration.

3. Core Engine Logic

3.1 PDF Compression Pipeline (Gzip)

Server-side Logic:

Buffer input stream from multi-part upload.

Apply zlib.gzipSync(buffer) for lossless compression.

Stream to S3 with Content-Encoding: gzip and ServerSideEncryption: AES256.

Log original_size and compressed_size for cost transparency reports.

3.2 Warehouse Mapping Logic

Grid System: Coordinates defined by Aisle (x) > Rack (y) > Shelf (z).

Occupancy Trigger: On every physical file movement, update the aggregate rack_occupancy cache to refresh the Heat Map color codes.

4. API Endpoint Specification (v1)

4.1 Digital Records

POST /api/v1/records/upload - Secure Gzip upload to S3.

GET /api/v1/patients/:uhid/history - Returns metadata and Signed URLs.

4.2 Warehouse Management

GET /api/v1/warehouse/map - Returns occupancy data for frontend visualization.

POST /api/v1/warehouse/check-out - Logs movement to a specific department.

GET /api/v1/labels/generate/:uhid - Returns QR payload for thermal printing.
