=============================================
DIGIFORT SCANNER - INSTALLATION
=============================================

1. Copy "DigifortScanner.exe" to your Desktop.

2. Run "RegisterProtocol.exe" (Right Click -> Run as Administrator).
   - You only need to do this ONCE.
   
3. Go to the Website -> Click "Desktop App".
   - It will open DigifortScanner automatically.

NOTE: Ensure both files are in the same folder during setup!
