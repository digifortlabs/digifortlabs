# Contributing to Digifort Labs

## Setup

### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `cp .env.example .env` (Configure your environment)
4. Run linting: `ruff check .`

### Frontend
1. `cd frontend`
2. `npm install`
3. Run dev server: `npm run dev`

## Code Style
- **Backend**: We use `ruff` for linting and formatting.
- **Frontend**: We use `eslint` and standard Prettier configuration.

## Pull Requests
- Update documentation if changing API behavior.
